# OTP Flutter Demo

Это минимальный пример приложения на Flutter с экранами ввода номера телефона и SMS-кода. После успешной верификации пользователь попадает на экран **Главная**.

## Запуск

Установите Flutter и выполните:

```bash
flutter pub get
flutter run
```

Приложение обращается к API `/send_otp` и `/verify_otp` (в коде используется базовый URL `https://example.com`). При необходимости измените адрес сервера.
